# cadastro-usuarios-python
Projeto desenvolvido para praticar lógica de programação, estruturas de dados e organização de código em Python.

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários cadastrados
- Menu interativo no terminal

## Tecnologias
- Python

## Como executar
1. Instale o Python
2. Execute o arquivo:
python main.py

## Objetivo
Praticar conceitos fundamentais de programação, como listas, dicionários, funções e repetição.

## Melhorias futuras
- Remover usuários
- Editar cadastro
- Salvar dados em arquivo
